## 📝 Descripción
Sistema completo de gestión de pedidos de agua, desarrollado con Flutter, Express.js y Docker. Permite registrar pedidos, desde una app móvil, los pedidos llegan en tiempo real, usando el protocolo de comunicación WS.

## 🚀 Tecnologías Usadas

- 🔧 **Flutter** – Aplicación móvil
- 🧠 **Express.js** – Backend RESTful
- 🐘 **PostgreSQL** – Base de datos relacional
- 🐳 **Docker + Docker Compose + Swarm** – Contenedores y despliegue
- 🐇 **RabbitMQ** – Mensajería en tiempo real (básica)
